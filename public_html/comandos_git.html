Comandos e dicas GIT

Principais comandos para trabalhar no dia a dia com o GIT e GitHub

Clonar um repositório remoto para o seu computador: >git clone https://github.com/autor_repositorio/nome_repositorio.git<

Adicionar um arquivo para ser rastreado pelo controle de versão: >git add nome_do_arquivo<

Adicionar todos os arquivos da pasta para serem rastreados pelo controle de versão: >git add .<

Commitar um arquivo alterado: "git commit -m >Comentário enxuto e preciso sobre a alteração"<

Subir os arquivos commitados no computador para o repositório remoto: >git push origin nome_da_branch<
